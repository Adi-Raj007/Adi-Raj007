### 🔄 COBOL to Python – Dataset Generation Research *(In Progress)*
> Deterministic + LLM hybrid pipeline for generating training data for COBOL-to-Python migration

- Researching automated generation of high-quality SFT (Supervised Fine-Tuning) datasets from legacy COBOL codebases.
- Generates 5 dataset types: `pic_reasoning`, `type_compatibility`, `memory_layout`, `call_graph`, and `data_division`.
- Building a compiler-style analysis layer — AST parser, symbol table, PIC analyzer, memory layout calculator, and control flow extractor — to eliminate LLM hallucination on deterministic facts.
- LLM handles only natural-language output; all structural facts are computed deterministically first.
- Pipeline: `read_cobol → preprocess → extract → validate → generate SFT examples`
- **Stack:** Python · Pydantic · LangChain · LLaMA3 · AST Parsing · pytest
